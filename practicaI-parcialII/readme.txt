Sistema de retiro de asignaturas

Luego de presentar la primera ronda de parciales, los estudiantes tienen la posibilidad de retirar
asignaturas en curso. Para automatizar este proceso, se desea que realice un formulario web donde se
recaude la siguiente información de un un estudiante: Nombre, Apellido, Cédula, Carrera que estudia,
número de semestre con mas materias inscritas. Y respecto a la asignatura a retirar debe solicitar:
nombre de la asignatura, código de la asignatura, unidades de crédito y un campo de texto para
especificar el motivo por el cual retirar la materia.
Una vez rellenado todos los campos, el estudiante presionará el botón con el texto Procesar
retiro para el diferido y la página debe cargar una nueva página donde deberá mostrar toda la
información del formulario para confirmar que la información fue enviada correctamente.
Requerimientos
1. La página debe tener el título: Sistema de retiro de asignaturas. Y debe mostrarse antes del formulario.
2. El formulario debe 2 conjuntos de campos, una para los datos del estudiante y otro para la asignatura.
Debe usar los tipos de campo correspondientes para cada tipo de datos (por ejemplo, tipo número para
las unidades de crédito)
3. El sitio debe incluir un archivo de CSS e implementar al menos 2 reglas donde se use selectores de
elementos, 2 reglas de identificadores (ids) y 2 reglas donde se usen clases.
4. El sitio debe implementar un archivo JS que realice alguna acción diferente de imprimir algo con alert o
registre un mensaje en la consola. Como idea, puede hacer que valide el envío del formulario o realice un
cambio al presionar un botón, entre otros.
5. Para el envío de formulario, los datos deben ser recibido en otra página que a través de PHP, muestre
el contenido solicitado.
